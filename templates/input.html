Format File CSV / Excel

  • Kolom wajib: Petugas, Tgl_Antaran_Pertama, Alamat, No_Resi, Nama
  • File ini hanya digunakan untuk menu Data Historis
  • Format tanggal: YYYY-MM-DD
  • Alamat harus lengkap dan akurat untuk geocoding
  • File dapat berupa .csv, .xlsx, atau .xls
  • Struktur kolom CSV harus sama dengan file Excel
